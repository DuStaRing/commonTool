## 什么是 commonTool？
commonTool，如你所见，它是一个小型整合型的工具类，它可以让我们脱离繁琐的开发流程，让开发变得**So easy!**

## 快速开始

- 引入依赖
```xml
<dependency>
  <groupId>com.zhouzifei</groupId>
  <artifactId>commonTool</artifactId>
  <version>1.0.1</version>
</dependency>
```
